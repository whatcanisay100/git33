# 1.markdown的标题  
###   markdown的标题是用若干数目的“#”再加空格，来表示字体的大小，以此来呈现主次  
# 2.markdown的段落格式  
### 通过末尾使用两个空格再加上回车，或者也可以在段落后使用一个空行来重新开始一个段落。  
### 通过“*”或“_”的数量来决定字体是粗是斜。

### 还可以通过三个以上的星号，减号，底线来建立一个分隔线。
